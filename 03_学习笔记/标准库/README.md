#### plantUML

网站链接：https://www.dumels.com/

这个网站可以分析一个 Go 项目，然后生成接口和结构体的 UML 图。有了这个图之后，基本上也就对项目整体关系有了一个基本概念，再读源码的话，相对来说会容易一些。
